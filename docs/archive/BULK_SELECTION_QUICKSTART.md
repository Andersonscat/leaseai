# ☑️ Массовый выбор - Быстрый старт

## 🎯 Что это?

Теперь вы можете **выбирать несколько объектов недвижимости** и **удалять их группой** вместо того, чтобы удалять по одному!

---

## 📸 Как это выглядит

### 1. Кнопка "Select" (справа от Filter)

```
┌─────────────────────────────────────────────┐
│  Properties                  [↕️] [Filter] [Select]  │
│  Manage your real estate listings          │
└─────────────────────────────────────────────┘
```

### 2. Режим выбора активен

```
┌─────────────────────────────────────────────┐
│  Properties                  [↕️] [Filter] [Cancel]  │
│                                     ↑ Синяя кнопка
└─────────────────────────────────────────────┘

┌──────────────────────────┐
│ ☑️ ← Чекбокс появляется   │
│ [Фото недвижимости]      │
│ $2,500/month             │
│ 123 Main St              │
│ 2 BD | 2 BA | 950 sq.ft  │
└──────────────────────────┘
```

### 3. Выбрано несколько объектов

```
         Плавающая панель внизу:
┌──────────────────────────────────────┐
│  ☑️ 3 selected | [Select All] [Delete] │
└──────────────────────────────────────┘
```

### 4. Модальное окно подтверждения

```
┌─────────────────────────────────────┐
│              🗑️                      │
│                                     │
│    Delete 3 Properties?             │
│                                     │
│    Are you sure you want to         │
│    delete 3 properties?             │
│                                     │
│  [Cancel]        [Delete All]       │
└─────────────────────────────────────┘
```

---

## 🚀 Как использовать (3 шага)

### Шаг 1: Включить режим
```
Нажмите кнопку "Select" → Появятся чекбоксы
```

### Шаг 2: Выбрать объекты
```
Вариант А: Кликайте на чекбоксы ☑️
Вариант Б: Кликайте на карточки (в режиме выбора)
Вариант В: Нажмите "Select All" внизу
```

### Шаг 3: Удалить
```
Нажмите красную кнопку "Delete" → Подтвердите → Готово!
```

---

## ⌨️ Быстрые действия

| Действие | Как сделать |
|----------|-------------|
| Выбрать все | Нажать "Select All" в нижней панели |
| Снять все | Нажать "Deselect All" |
| Выбрать один | Клик на чекбокс или карточку |
| Снять один | Клик на выбранный чекбокс |
| Выйти | Нажать "Cancel" (синяя кнопка) |
| Удалить | Красная кнопка "Delete" → Подтвердить |

---

## 💡 Полезные советы

### 1. Выбрать все кроме одного
```
1. Select All
2. Снять галочку с ненужного
```

### 2. Быстро найти и удалить
```
1. Введите в поиск (например "Seattle")
2. Нажмите Select
3. Выберите все результаты
4. Удалите группой
```

### 3. Отменить случайный выбор
```
Просто нажмите "Cancel" - всё сбросится
```

## ⚡ Экономия времени

**До (без массового выбора):**
```
Удалить 10 объектов:
- Открыть property → Удалить → Подтвердить = 3 клика
- 10 объектов × 3 = 30 кликов
- Примерно 2-3 минуты
```

**После (с массовым выбором):**
```
Удалить 10 объектов:
- Select → 10 кликов → Delete → Confirm = 13 кликов
- Примерно 30 секунд
```

**Экономия: 57% времени!** 🚀

## ❌ Что НЕ работает в режиме выбора

- ❌ Клик на карточку НЕ открывает property (только выбирает)
- ❌ Нельзя перейти на страницу property
- ❌ Нажмите "Cancel" чтобы вернуться к обычному режиму

## ⚠️ Важно знать

### Безопасность
- ✅ Модальное окно спросит подтверждение
- ✅ Показывает сколько объектов будет удалено
- ✅ Можно отменить до самого конца
- ✅ Кнопка блокируется во время удаления

### Восстановление
- Удаление = Soft Delete (помечается как deleted_at)
- Можно восстановить через SQL (пока нет UI)
- В будущем добавим вкладку "Archived"

### Производительность
- Удаляет все объекты параллельно (быстро)
- Список автоматически обновляется
- Toast показывает результат

## 🐛 Если что-то не работает

### Кнопка Select не появляется
→ Проверьте что вы на вкладке "Properties"

### Чекбоксы не появляются
→ Нажали ли вы на кнопку "Select"?

### Клик на карточку открывает property
→ Вы не в режиме выбора. Нажмите "Select" сначала

### Нижняя панель не появляется
→ Выберите хотя бы 1 property

### Удаление не работает
→ Проверьте:
- Вы залогинены?
- Properties принадлежат вам?
- Есть интернет соединение?
