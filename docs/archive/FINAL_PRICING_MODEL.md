## 🔄 КАК ЭТО РАБОТАЕТ:

### **Контакты:**
```typescript
// При добавлении нового лида
const check = await canAddContact(userId);

if (!check.allowed) {
  return {
    error: 'Contact limit reached',
    message: `You have ${check.current}/${check.limit} contacts. 
              Delete archived contacts or upgrade.`,
    upgradeUrl: '/billing'
  };
}

// Добавляем контакт
await createContact();
```

### **AI Ассистент:**
```typescript
// При отправке сообщения в чат
const check = await canUseAIAssistant(userId);

if (!check.allowed) {
  return {
    error: 'Daily limit reached',
    message: `Used ${check.used}/${check.limit} messages today.
              Resets in ${formatTime(check.resetsAt)}`,
    resetsAt: check.resetsAt
  };
}

// Отправляем сообщение AI
await sendToAI();
await incrementDailyUsage(userId, 'aiAssistant');
```

### **Автоответы:**
```typescript
// При получении нового email от лида
const check = await canUseAutoResponse(userId);

if (check.allowed) {
  // Отправляем автоответ
  await sendAutoResponse();
  await incrementDailyUsage(userId, 'autoResponse');
} else {
  // Пропускаем автоответ, просто добавляем в inbox
  console.log('Auto-response limit reached, skipping');
}
```
