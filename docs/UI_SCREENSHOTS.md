## 🎨 Design System

### Colors
- **Primary**: Blue (#2563EB) - Buttons, links, brand
- **Background**: White/Gray-50
- **Text**: Gray-900 (headings), Gray-600 (body)
- **Success**: Green-500 (checkmarks)
- **Border**: Gray-300

### Typography
- **Font**: Inter (Google Font)
- **Headings**: Bold, 24-48px
- **Body**: Regular, 14-16px
- **Small**: 12-14px

### Components
- **Button**: Rounded, padding, hover states
- **Card**: Shadow, border, rounded corners
- **Input**: Border focus, validation

### Spacing
- **Container**: Max-width 1200px, centered
- **Section**: Padding 64-96px vertical
- **Cards**: Gap 24-32px

### Icons
- **Library**: Lucide React
- **Size**: 16-24px standard, 48px hero
- **Style**: Outline, consistent weight

### Responsive
- **Mobile**: Single column, stacked
- **Tablet**: 2 columns where applicable
- **Desktop**: 3 columns, full layout

## 🔄 User Flows

### New User Signup
```
Landing → Sign Up → Email Verify → Dashboard
```

### Existing User Login
```
Landing → Sign In → Dashboard
```

### Upgrade Flow
```
Dashboard → Billing → Stripe Checkout → Payment → Dashboard (success)
```

### Sign Out
```
Dashboard → Profile Menu → Sign Out → Landing
```
